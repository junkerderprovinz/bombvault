Publish-polish release — BombVault is now on the Unraid **Community Applications** feed.

## Added
- **Browser-tab icon:** the BombVault logo now shows as the favicon in your browser tab (was the default icon).

## Changed
- **Community Applications listing:** the CA description now leads with the claim and positions BombVault as the best (and only) backup tool you need for Unraid, covering all three domains (containers, VMs, flash) plus off-site repos, retention, file-level restore, integrity checks and pre/post-backup hooks.

## Install
Search **BombVault** in Unraid → **Apps** (Community Applications). The image `ghcr.io/junkerderprovinz/bombvault` is public.

Image: `ghcr.io/junkerderprovinz/bombvault:1.3.3` (also `:1`, `:1.3`, `:latest`), multi-arch (amd64 + arm64).

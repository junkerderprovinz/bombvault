Adds an opt-out for the scheduled off-site DR drill, to save egress on metered clouds.

## Added

- **Turn off the scheduled off-site disaster-recovery drill.** (#37, thanks @BaukeZwart) The off-site DR drill restores your whole off-site snapshot on a schedule to prove remote recovery, which re-downloads the entire backup each run and costs egress on metered clouds (for example Backblaze B2). A new **Scheduled off-site DR drill** toggle (Settings → Automatic restore checks, Advanced) lets you turn it off while keeping the free scheduled local integrity check. With it off:
  - the dashboard's off-site restorability shows a neutral **"Off-site DR: manual only"** state instead of nagging, and the posture chip no longer counts a missing scheduled DR against you;
  - you can still run the off-site DR check on demand with the **Run off-site DR check** button on the dashboard;
  - an honest result is still shown: a drill that genuinely fails still reads red, and a passing run (scheduled or manual) still turns it green.
  - Default is on, so existing setups are unchanged until you opt out.

Adds a live indicator showing which domain is currently replicating off-site. Thanks **@manilx** for the suggestion.

## Added
- **Off-site replication indicator.** While a domain's snapshots are being copied off-site, an "off-site replication running" indicator shows on that domain's page (Containers / VMs / Flash) and on the Dashboard (labelled per domain), so you can see at a glance which one is running. It's an active/indeterminate indicator, not a percentage bar: `restic copy` exposes no machine-readable progress (only `backup` does), so a true percentage isn't available — the indicator reflects start and finish.

Image: `ghcr.io/junkerderprovinz/bombvault:2.5.0` (also `:2`, `:2.5`, `:latest`) and `docker.io/junkerderprovinz/bombvault:2.5.0`, multi-arch (amd64 + arm64).

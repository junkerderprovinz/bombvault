A friendlier UI: an Advanced toggle keeps the app clean by default, schedules read in plain language, and the version + bug-report link move out of the sidebar.

## Added
- **Advanced mode.** The interface now shows only the essentials by default (back up, restore, schedule, paths, encryption, stacks). Flip the **Advanced** toggle in the sidebar to reveal the expert controls across the whole app — retention, off-site copy, pre/post hooks, file-level restore, notifications, Prometheus metrics, integrity/maintenance tools, restore-verification drills and more. It's a per-browser preference, off by default, so newcomers get a clean UI and power users still get everything. (VM-over-SSH stays visible whenever the VMs domain is enabled, so it never blocks VM backups.)
- **Version and "Report a bug" links.** The running version links to the releases page, and a new **Report a bug** link opens the GitHub issues page. (#23, thanks @manilx)

## Changed
- **Schedules read in plain language.** The dashboard protection-status card and the schedule builder now show e.g. **"every 3 days at 4:00"** instead of the raw `everyN 3 04:00` — localized, with the time without a leading zero and weekday names in your language.
- **Cleaner sidebar.** The version and the "Report a bug" link moved out of the sidebar into a small footer at the very end of **Settings**, keeping the sidebar to just navigation and preferences.
- The README footer now cross-links to the other junkerderprovinz Unraid apps.
